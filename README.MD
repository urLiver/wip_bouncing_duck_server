# Bouncing Duck Server Files for Plutonium IW5
This repository contains the Files that can be used to host a custom Infected Bounce Server for Plutonium IW5

Original Inspired by Classic Infected hosted by GilletteClan

## How to use
Put mods and scripts folder on your server

## Known Bugs:
* Unless you got a custom .ff that unloads images these maps will crash with the mod.ff mp_compact and mp_radar
